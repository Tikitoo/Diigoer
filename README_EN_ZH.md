## Diigoer
Digoer 是一个Diigo Android 客户端，实现了登录，查看书签，对书签的分类，查找，分享（也可分享其他URL 到Diigo），缓存，设置主题等功能：
- [http://www.github.com/Tikitoo/Diigoer]()
